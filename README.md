# 地理学与环境科学期刊投稿指南

[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

> 📚 一份全面的国内地理学、环境生态学、遥感学、地理信息、测绘领域期刊投稿指南

## 📖 项目简介

本项目旨在为地理学、环境生态学、遥感学、地理信息系统（GIS）和测绘学领域的研究人员提供：

- ✅ **国内外核心期刊介绍**：涵盖一二区权威期刊
- ✅ **详细投稿信息**：影响因子、审稿周期、投稿网址
- ✅ **实用模板工具**：Cover Letter、审稿回复模板
- ✅ **学术资源汇总**：写作工具、查询平台、学习资源
- ✅ **辅助脚本**：投稿材料检查、时间线规划

## 🗂️ 目录结构

## 🚀 快速开始

### 1. 查看期刊指南

直接阅读核心文档了解各期刊详细信息：

```bash
# 在浏览器中打开
open docs/国内地理学环境生态学期刊投稿指南.md

# 或使用命令行查看
cat docs/国内地理学环境生态学期刊投稿指南.md
```

### 2. 使用投稿模板

- **Cover Letter**: `templates/cover_letter_template.md`
- **审稿回复**: `templates/review_response_template.md`

### 3. 运行辅助工具

```bash
# 运行投稿检查工具
python scripts/submission_helper.py
```

### 4. 查阅学术资源

访问 `resources/academic_resources.md` 获取：
- 期刊查询网站
- 学术写作工具
- 科研绘图软件
- 在线学习平台
